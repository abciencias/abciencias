# ABCiências

Portal oficial da ABCiências — Mostra de Ciências da Educação Básica.

## Estrutura

- `index.html`: página inicial
- `css/style.css`: estilos do site
- `js/script.js`: recursos de interação
- `paginas/`: páginas internas
- `documentos/`: editais e arquivos em PDF
- `imagens/`: logotipos, banners e fotos
