# Portal ABCiências v2.0 completa

Versão com base no edital 01/2026 e nas informações institucionais da ABCCEL.

## Novidades
- Página inicial completa
- Menu com Edições e Galeria
- Página da Edição 2025
- Página de Galeria
- Página do Edital 2026
- Documento do edital em `documentos/edital-abciencias-2026.docx`

## Como adicionar fotos
1. Coloque as fotos em `imagens/edicao-2025/`.
2. Recomenda-se nomes simples: `foto-01.jpg`, `foto-02.jpg`, etc.
3. Depois substitua os blocos `foto-placeholder` nas páginas `galeria.html` e `edicao-2025.html` por imagens reais.
